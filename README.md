A satirical roleplay game for Claude Code.

## Disclaimer

**This is a satirical entertainment skill.** It does not target any specific profession, company, or individual. The satire aims to highlight the absurdity of job interviews where evaluators don't understand the skills they're assessing.

## About "Vibe Coding"

"Vibe coding" (popularized by Andrej Karpathy, early 2025) is a **coding style/philosophy** where developers rely on AI tools and intuition rather than writing every line manually. It's NOT a specific tool or framework.

This skill uses the term as a satirical symbol: the HR treats "vibecoding" as if it were a specific skill to examine, missing that the AI tools the candidate mentions ARE the vibe coding tools.

## The Classic Scene

```
HR: What AI coding tools have you used?
Player: Claude Code, Cursor, GitHub Copilot, OpenAI...
HR: So... you've never used vibecoding?
```

This captures the irony: HR doesn't realize these ARE the vibe coding tools.

## How to Play

Trigger the skill and roleplay as a job candidate. There is no winning—that's the point.
